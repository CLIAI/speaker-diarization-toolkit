# Speechmatics API Reference Overview

Speechmatics provides flexible communication protocols for speech processing applications. According to their documentation, the platform offers "REST and Websocket APIs for transcription and voice AI applications."

## Available API Categories

**Transcription Services:**

* Real-time audio transcription via WebSocket connections
* Batch processing through REST endpoints for job creation, management, and transcript retrieval
* Job monitoring capabilities including usage statistics

**Voice AI:**

* Agent conversation initiation through WebSocket protocol

## Key API Endpoints

The platform includes functionality to:

* Create transcription jobs
* Retrieve job details
* Delete jobs
* List all jobs
* Obtain transcripts
* Access usage analytics
* Initiate voice AI conversations programmatically

## Access Requirements

Authentication requires an API key, obtainable through the Speechmatics portal settings.

The documentation indicates this is a developer-focused platform supporting both synchronous (real-time) and asynchronous (batch) transcription workflows, alongside conversational AI capabilities built on WebSocket architecture.

## Platform Overview

Speechmatics provides "enterprise-grade APIs for speech-to-text, text-to-speech, and voice AI agents." The platform enables developers to integrate voice technology without managing underlying infrastructure.

## Core Capabilities

**Main Features:**

* Real-time transcription from live audio streams
* Batch processing for recorded audio files
* Voice AI agent development via Voice SDK
* Text-to-speech conversion
* Flexible deployment (SaaS or on-premises)
